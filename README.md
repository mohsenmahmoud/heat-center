# هيت سنتر — Heat Center Website Redesign

موقع تسويقي جديد بالكامل لشركة **هيت سنتر**، مبني بـ React + TypeScript +
Vite + Tailwind CSS v4، بهوية بصرية مميزة (Ember/Ink/Aqua) ومحتوى معاد
صياغته بالكامل باللغة العربية.

## التشغيل محليًا

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
npm run preview
```

## بنية المشروع

- `src/data/content.ts` — كل نصوص ومحتوى الموقع (خدمات، إحصائيات، أسئلة شائعة، بيانات التواصل) فى مكان واحد لسهولة التعديل.
- `src/components/layout` — الهيدر (Navbar)، الفوتر، زر واتساب العائم.
- `src/components/ui` — مكونات واجهة قابلة لإعادة الاستخدام (أزرار، بطاقات، عدّاد إحصائيات، أكورديون الأسئلة الشائعة).
- `src/pages` — الصفحات: الرئيسية، من نحن، خدماتنا، أعمالنا، الأسئلة الشائعة، تواصل معنا.

## ملاحظات قبل الإطلاق

- بيانات التواصل (الهاتف، واتساب، البريد، روابط السوشيال ميديا) فى `src/data/content.ts` هى بيانات مؤقتة (placeholder) ويجب استبدالها بالبيانات الحقيقية.
- قسم معرض الأعمال فى صفحة "أعمالنا" يحتاج صور حقيقية من المشروعات المنفذة بدلًا من العناصر التجريبية الحالية.
- نموذج التواصل فى صفحة "تواصل معنا" حاليًا يعرض رسالة نجاح فى المتصفح فقط، ويحتاج ربطًا بخدمة بريد إلكتروني أو API لإرسال الطلبات فعليًا.
